# 2장 변수와 수식
# 1. 변수
데이터를 저장하는 공간으로 값을 저장하고 변경할 수 있다.
# 2. 자료형
* 정수(int)
* 실수(float)
* 문자열(str)
# 3. 입력과 출력
* input() : 사용자 입력
* print() : 결과 출력
# 4. 형 변환
자료형을 다른 자료형으로 변환한다.
* int()
* float()
* str()
# 5. 주의해야 할 점
* 변수명은 숫자로 시작할 수 없다.
* 예약어는 변수명으로 사용할 수 없다.
# 📌 핵심 정리
* 변수는 데이터를 저장하는 공간이다.
* 자료형에는 정수, 실수, 문자열 등이 있다.
* input()으로 입력받고 print()로 출력한다.
