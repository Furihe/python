# 6장 파이썬 자료구조 1 (리스트)
# 1. 리스트의 개념

* 리스트는 여러 개의 데이터를 하나의 변수에 저장할 수 있는 자료구조이다. 같은 자료형뿐만 아니라 서로 다른 자료형의 데이터도 함께 저장할 수 있다.

# 2. 리스트 생성

* 대괄호 []를 사용하여 리스트를 생성한다.

* 여러 개의 값을 저장할 수 있다.
* 각 요소는 쉼표(,)로 구분한다.
# 3. 인덱스

* 리스트의 각 요소에는 위치 번호가 있다.

* 첫 번째 요소의 인덱스는 0이다.
* 음수 인덱스를 사용하면 뒤에서부터 접근할 수 있다.
# 4. 리스트 연산

* 리스트의 요소를 추가, 수정, 삭제할 수 있다.

* append() : 요소 추가
* insert() : 원하는 위치에 요소 삽입
* remove() : 요소 삭제
* pop() : 요소를 꺼내며 삭제
# 5. 슬라이싱

* 리스트의 일부 요소를 추출하는 기능이다.

* 시작 위치와 끝 위치를 지정하여 사용한다.
* 끝 위치는 포함되지 않는다.
# 6. 리스트 반복 처리

반복문을 이용하여 리스트의 모든 요소를 처리할 수 있다.

* for 문과 함께 자주 사용된다.
* 리스트의 데이터를 순서대로 접근할 수 있다.
# 7. 리스트 함축(List Comprehension)

* 반복문과 조건문을 사용하여 새로운 리스트를 간단하게 생성하는 방법이다.

* 코드가 간결해진다.
* 리스트 생성에 자주 사용된다.
# 8. 주의해야 할 점
* 인덱스 범위를 벗어나면 오류가 발생한다.
* 삭제된 요소에는 접근할 수 없다.
* 리스트와 문자열의 인덱스는 0부터 시작한다.
# 📌 핵심 정리
* 리스트는 여러 개의 데이터를 저장하는 자료구조이다.
* 인덱스를 사용하여 요소에 접근할 수 있다.
* append(), insert(), remove(), pop() 등을 사용하여 데이터를 관리한다.
* 슬라이싱으로 원하는 부분만 추출할 수 있다.
* 반복문과 함께 사용하면 리스트의 모든 요소를 처리할 수 있다.
* 리스트 함축을 이용하면 간결하게 리스트를 생성할 수 있다.
