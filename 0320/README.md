# 3장 조건문
# 1. 제어문과 제어 구조

프로그램의 실행 흐름을 제어하는 문장을 제어문이라고 한다.

순차 구조 : 위에서 아래로 순서대로 실행
선택 구조 : 조건에 따라 다른 코드 실행
반복 구조 : 같은 코드를 여러 번 실행
# 2. 관계 연산자

두 값을 비교하여 참(True) 또는 거짓(False)을 반환하는 연산자이다.

# 3. 논리 연산자

여러 조건을 함께 사용할 때 사용하는 연산자이다.

and : 모든 조건이 참일 때 참
or : 조건 중 하나라도 참이면 참
not : 결과를 반대로 바꿈
# 4. 조건문(if)

조건식의 결과에 따라 실행할 코드를 결정한다.

if : 조건이 참일 때 실행
if-else : 참과 거짓에 따라 다른 코드 실행
if-elif-else : 여러 조건 중 하나를 선택하여 실행
# 5. 중첩 조건문

조건문 안에 또 다른 조건문을 넣어 사용할 수 있다. 복잡한 조건을 처리할 때 사용한다.

# 6. 조건 연산자

간단한 조건문을 한 줄로 표현할 수 있다.

값1 if 조건식 else 값2
# 7. 주의해야 할 점
조건식 뒤에는 반드시 콜론(:)을 붙여야 한다.
조건문 내부의 코드는 들여쓰기를 해야 한다.
같은 블록의 들여쓰기 깊이는 동일해야 한다.
관계 연산자 =와 ==를 혼동하지 않도록 주의한다.
# 📌 핵심 정리
조건문은 조건의 참(True), 거짓(False)에 따라 실행 흐름을 결정한다.
관계 연산자와 논리 연산자를 이용해 조건식을 작성한다.
if, if-else, if-elif-else를 사용하여 다양한 조건을 처리할 수 있다.
조건문에서는 콜론(:)과 들여쓰기가 매우 중요하다.
