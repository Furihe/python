# 4장 반복문
# 1. 반복문의 개념

반복문은 같은 작업을 여러 번 수행할 때 사용하는 제어문이다. 코드를 반복해서 작성하지 않고 효율적으로 프로그램을 작성할 수 있다.

# 2. for 반복문

정해진 횟수만큼 반복할 때 사용한다.

range() 함수와 함께 자주 사용한다.
리스트, 문자열 등의 데이터를 순서대로 처리할 수 있다.
# 3. range() 함수

일정한 범위의 숫자를 생성하는 함수이다.

ex)
* range(5) : 0~4
* range(1, 6) : 1~5
* range(1, 10, 2) : 1, 3, 5, 7, 9
# 4. while 반복문

조건이 참(True)인 동안 계속 반복한다.

반복 횟수가 정해져 있지 않을 때 사용한다.
조건이 거짓(False)이 되면 반복을 종료한다.
# 5. 중첩 반복문

반복문 안에 또 다른 반복문을 사용하는 방식이다.

# 6. 반복문 제어

반복문의 실행을 제어하는 명령어이다.

* break : 반복문을 즉시 종료
* continue : 현재 반복을 건너뛰고 다음 반복 실행
# 7. 무한 반복문

종료 조건이 없으면 반복문이 계속 실행될 수 있다.

* while True: 형태로 작성한다.
반드시 종료 조건을 함께 고려해야 한다.
# 8. 주의해야 할 점
* 반복문 뒤에는 반드시 콜론(:)을 붙여야 한다.
* 반복문 내부는 들여쓰기를 해야 한다.
* while 문은 종료 조건을 잘 설정해야 한다.
무한 반복문이 발생하지 않도록 주의한다.
# 📌 핵심 정리
반복문은 같은 작업을 여러 번 수행할 때 사용한다.
* for 문은 정해진 횟수만큼 반복한다.
* while 문은 조건이 참인 동안 반복한다.
* break는 반복을 종료하고 continue는 다음 반복으로 넘어간다.
* 반복문에서는 콜론(:)과 들여쓰기가 중요하며, 무한 반복문에 주의해야 한다.
