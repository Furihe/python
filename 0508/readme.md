# 7장 파이썬 자료구조 2
# 1. 튜플(Tuple)

튜플은 여러 개의 데이터를 저장하는 자료구조이다.

소괄호 ()를 사용한다.
리스트와 비슷하지만 값을 수정할 수 없다.
데이터가 변경되지 않아야 할 때 사용한다.
# 2. 튜플 연산

튜플도 인덱싱과 슬라이싱이 가능하다.

인덱스를 이용해 요소에 접근할 수 있다.
요소의 추가, 삭제, 수정은 불가능하다.
# 3. 딕셔너리(Dictionary)

딕셔너리는 키(Key)와 값(Value)의 쌍으로 데이터를 저장하는 자료구조이다.

중괄호 {}를 사용한다.
키를 이용해 값을 빠르게 찾을 수 있다.
키는 중복될 수 없다.
# 4. 딕셔너리 활용

딕셔너리에 데이터를 추가, 수정, 삭제할 수 있다.

딕셔너리[키] = 값 : 추가 및 수정
* del : 삭제
* keys() : 키 조회
* values() : 값 조회
* items() : 키와 값 조회
# 5. 집합(Set)

집합은 중복되지 않는 데이터를 저장하는 자료구조이다.

중괄호 {}를 사용한다.
중복된 값은 자동으로 제거된다.
순서가 없다.
# 6. 집합 연산

집합은 수학의 집합 연산을 지원한다.

* 합집합 : |
* 교집합 : &
* 차집합 : -
# 7. 자료구조 비교
* 리스트(List) : 수정 가능, 순서 있음
* 튜플(Tuple) : 수정 불가, 순서 있음
* 딕셔너리(Dictionary) : 키와 값으로 저장
* 집합(Set) : 중복 불가, 순서 없음
# 8. 주의해야 할 점
* 튜플은 생성 후 값을 변경할 수 없다.
* 딕셔너리의 키는 중복될 수 없다.
* 집합은 인덱스를 사용할 수 없다.
* 집합은 데이터의 저장 순서를 보장하지 않는다.
# 📌 핵심 정리
* 튜플은 수정이 불가능한 자료구조이다.
* 딕셔너리는 키와 값의 쌍으로 데이터를 저장한다.
* 집합은 중복을 허용하지 않는다.
* 딕셔너리는 빠른 데이터 검색에 유용하다.
* 집합은 합집합, 교집합, 차집합 연산을 지원한다.
* 리스트, 튜플, 딕셔너리, 집합의 특징을 구분할 수 있어야 한다.
