# 5장 함수
# 1. 함수의 개념

함수는 특정 작업을 수행하는 명령문들의 집합이다. 자주 사용하는 코드를 함수로 만들어 두면 재사용이 가능하며 프로그램을 효율적으로 작성할 수 있다.

# 2. 함수 정의

함수는 def 키워드를 사용하여 정의한다.

함수 이름을 정하고 기능을 구현한다.
필요할 때 여러 번 호출하여 사용할 수 있다.
# 3. 함수 호출

정의된 함수를 실행하는 것을 함수 호출이라고 한다.

함수 이름 뒤에 괄호 ()를 사용한다.
호출해야 함수 내부 코드가 실행된다.
# 4. 매개변수와 인수

함수에 값을 전달하기 위해 사용한다.

매개변수(Parameter) : 함수 정의 시 사용하는 변수
인수(Argument) : 함수 호출 시 전달하는 실제 값
# 5. 반환값

함수의 실행 결과를 호출한 곳으로 돌려주는 값이다.

return 문을 사용한다.
반환값은 변수에 저장하거나 다른 연산에 사용할 수 있다.
# 6. 지역 변수와 전역 변수

변수의 사용 범위에 따라 구분된다.

지역 변수 : 함수 내부에서만 사용 가능
전역 변수 : 프로그램 전체에서 사용 가능
# 7. 내장 함수

파이썬에서 기본적으로 제공하는 함수이다.

print()
input()
len()
max()
min()
sum()
등이 있다.

# 8. 주의해야 할 점
* 함수 정의 시 def 뒤에 함수 이름과 괄호를 작성해야 한다.
* 함수 정의 뒤에는 반드시 콜론(:)을 붙인다.
* 함수 내부 코드는 들여쓰기해야 한다.
* 함수를 정의한 후 반드시 호출해야 실행된다.
# 📌 핵심 정리
* 함수는 특정 기능을 수행하는 코드의 묶음이다.
* def를 사용하여 함수를 정의한다.
* 매개변수와 인수를 통해 데이터를 전달할 수 있다.
* return을 사용하여 결과를 반환할 수 있다.
* 지역 변수와 전역 변수의 차이를 이해해야 한다.
* 함수를 사용하면 코드의 재사용성과 가독성을 높일 수 있다.
